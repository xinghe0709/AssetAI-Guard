## In scope (MVP)

- Locations, Assets, Load Capacities stored in database
- Dependent dropdowns: Location → Asset
- Equipment selection that determines load parameter label + fixed metric
- Compliance check (pass/fail) by comparing input value to the correct asset capacity using the provided mapping
- **Saving records of compliance checks** (store user inputs + computed result + timestamp + which asset capacity was referenced)
- Result includes explanation and references which capacity was used
